Profiterole is a student project.
The main goal of this project is learning tehnologies
such as Spring(c) Hibernate(c) and other production java technologies
which are used in comerxial development.

Profiterole is a "kichen calculator".
You can create dish and than select how many peolple you want to feed.
Project will calculate how many component's you need to prepare you dinner.
